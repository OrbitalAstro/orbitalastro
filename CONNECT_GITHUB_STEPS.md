# Step-by-Step: Connect GitHub to Vercel

## Quick Steps

1. **The Vercel project page should have opened in your browser**
   - If not, go to: https://vercel.com/jo-divers-projects/orbitalastro-api/settings/git

2. **Click "Connect Git Repository"** button
   - This will be in the Git section of the settings page

3. **Authorize GitHub** (if prompted)
   - You'll be redirected to GitHub
   - Click "Authorize Vercel" or "Install Vercel"
   - Grant access to your repositories (or just the OrbitalAstro organization)

4. **Select Your Repository**
   - Choose: `OrbitalAstro/orbitalastro`
   - Click "Connect" or "Import"

5. **Wait for Connection**
   - Vercel will verify the connection
   - You should see a success message
