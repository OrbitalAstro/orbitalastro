# 📍 Où trouver SQL Editor dans Supabase

## ✅ Oui, c'est dans le menu de gauche de Supabase !

### 🎯 Emplacement exact

Une fois que vous êtes connecté à Supabase et que vous avez sélectionné votre projet, vous verrez un **menu vertical à gauche** avec plusieurs options.

### 📋 Menu de gauche typique dans Supabase :

```
┌─────────────────────────┐
│  [Logo Supabase]        │
│                         │
│  📊 Table Editor        │
│  📝 SQL Editor    ← ICI │
│  🔍 Database            │
│  🔐 Authentication      │
│  📧 Storage             │
│  ⚙️ Settings            │
│  📈 Logs                │
│  ...                    │
└─────────────────────────┘
```

### 🔍 Comment le reconnaître

**"SQL Editor"** peut apparaître comme :
- **"SQL Editor"** (texte)
- Icône `</>` (chevrons avec slash)
- Icône de code `{}`
- Parfois dans une section "Development" ou "Database"

### 📍 Étapes précises

1. **Allez sur** [https://supabase.com](https://supabase.com)
2. **Connectez-vous** avec votre compte
3. **Cliquez sur votre projet** `orbitalastro` (dans la liste des projets)
4. **Regardez le menu de gauche** (barre verticale)
5. **Cherchez "SQL Editor"** dans cette liste
6. **Cliquez dessus**

### 🎨 À quoi ça ressemble visuellement

Quand vous cliquez sur "SQL Editor", vous verrez :
- Un éditeur de texte au centre (comme un bloc-notes)
- Un bouton "New query" en haut
- Un bouton "Run" en bas à droite
- Une zone pour coller votre code SQL

### ⚠️ Si vous ne le voyez pas

1. **Vérifiez que vous êtes dans votre projet** (pas sur la page d'accueil)
2. **Le menu de gauche devrait être visible** (si vous êtes sur mobile, il peut être caché - cherchez un menu hamburger ☰)
3. **Faites défiler** le menu de gauche au cas où il y aurait beaucoup d'options
4. **Cherchez aussi** dans une section "Database" ou "Development"

### 🆘 Alternative : Via l'URL directe

Si vous avez du mal à trouver, vous pouvez aussi aller directement à :
```
https://supabase.com/dashboard/project/[VOTRE_PROJECT_ID]/sql/new
```

(Remplacez `[VOTRE_PROJECT_ID]` par l'ID de votre projet)
